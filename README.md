Computer Vision Reinforcement Learning Basketball Game Bot

This project implements a reinforcement learning agent that plays a browser-based basketball game using computer vision for state observation and keyboard automation for action execution.

Project Structure:
computer_vision/ - YOLO models and screen capture logic
reinforcement_learning/ - RL environment and agent definitions
automation/ - Keyboard control utilities
utilities/ - Helper functions
models/ - Saved model weights
datasets/ - Training data
documentation/ - Project docs
